  • The Window Object
    • The global object in the browser environment is the wwindow object, this means that any global variable created is actually a property of the window object. In general you should not refer to the window object when calling these variables.
    • window.location property is an object that contains information about the URL of the current page. This is a read/write property.
    • window.history property can be used to access information about any previously visited pages in the current browser session.
    • window.screen object contains information about the screen the browser is displayed on.
    • Document object has properties and methods that deal with the page that has been loaded into the window.
    • windows.setTimeout() method accepts a callback to a function as its first paramter and a number of milliseconds as its second parameter.
  • HTML5 APIs
    • Data attribute is a way of embedding data in a web page using custom attributes that are ignored by the browser. They’re also private to a page, so are not intended to be used by an external service – their sole purpose is to be used by a JavaScript program. This means they’re perfect for adding data that can be used as a hook that the program utilizes to access information about a particular element on the page.
    • Names of the data- attribues can be decided by the developer, but must follow the following format
      • Start with: data-
      • Contain only lowercase letters, numbers, hyphens, dots, colons or underscores
      • Include an optional string value
    • Websocket is a new protocol that allows two-way communication with a server – also known as push messaging. This means that a connection is kept open and responses are ‘pushed’ to the client as soon as they are received.
    • Canvas element was introduced to allow graphics to be drawn onto a web page in real time using Javascript. This is used in many things including charts.js
