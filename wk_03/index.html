  • Ch. 5 - Objects
    • Object literals are a distinguishing feature of the JavaScript language, as they allow objects to be created quickly without the need for defining a class. They also provide a useful way of organizing your code without polluting the global namespace.
    • Object 
      •             
                const superman = {
                    name: 'Superman',
                    'real name': 'Clark Kent',
                    height: 75,
                    weight: 235,
                    hero: true,
                    villain: false,
                    allies: ['Batman','Supergirl','Superboy'],
                    fly() {
                        return 'Up, up and away!';
                    }
                };
      • Each property is a key-value pair separated by commas
      • The property 'real name' in the example above needs to be quoted because it contains a space. Another way to name it is real_name or realName.
      • All properties are mutable which means they can be changed, removed, added even if it was declared using const.
      • Accessing Properties: superman.name or superman['name']
      • Dot notation is more common, but bracket notation allows you to access properties with nonstandard names and also lets you evaluate an expression and use it as the property keys
      • Computed Properties were made available in ES6
      • Finding all the properties of an Object: 
        •         
            for(const key in superman) {
                console.log(key + ": " + superman[key]);
            }
            << "name: Superman"
            << "real name: Clark Kent"
            << "height: 75"
            << "weight: 235"
            << "hero: true"
            << "villain: false"
            << "allies: Batman,Supergirl,Superboy"
            << "fly: function (){
                console.log(\"Up, up and away!\");
            }"
      • Adding Properties: superman.city = 'Metropolis';
      • Removing properties: delete superman.fly

    • Date Object
      • This creates the current date and time: const today = new Date();
      • The epoch is 1st January 1970 and is used as a reference point in time to measure dates.
      • date.getDay() returns the day as a number
      • date.getMonth() returns the month as a number
      • date.getFullYear() returns the year as a number
      • date.getTime() returns a timestamp representing the number of milliseconds since the Epoch
    • Regular Expression (RegEx)
      • const vowels = /[aeiou]/
      • All uppercase letters: /[A-Z]/
      • The digits 0-9 can be represented as: /[0-9]/
      • Everything except a capital letter: /[^A-Z]/
  • Chapter 6 - Document Object Model (DOM)
    • The DOM treats everything on a web page as a node. HTML tags, the text inside these tags, even the attributes of a tag are all nodes. The HTML tag is the root node, and every other part of the document is a child node of this.
    • document.getElementById()
    • document.getElementsByTagName()
    • document.getElementsByClassName()
    • document.querySelector()
      • by id: document.querySelectorAll('.hero');
    • document.querySelectorAll()
      • by class: document.querySelectorAll('.hero');
    • childNodes
    • children
    • firstChild
    • lastChild
    • nextSibling (for parent node)
    • previousSibling
