Week 10

Block 2 Challenge

  • Purpose
    • I want to build a tool that allows a user to change the website banner as they see fit. The banner will include a logo, business name, slogan, and 3 other images that they can change. The user can change all images, text and colors. They will not have the ability to change the set layout, but can adapt everything within that design how they want it to look.
  • Audience
    • website owner
  • Data Sources
    • local JSON file
  • Initial Module List
    • logo
    • Title and slogan
    • Images and borders
    • All data saved in local JSON file
  • Wireframes
  • Colors/Typography/Specific Element styling
    • The color, typography, and sizing will all be editable by the user to change the site within the framework designed. This means that each word can be a different color and that the size, font, and font weight of the title and slogan will be editable as well.
  • Schedule
    • Small, medium, and large layout done by end of week 10
    • Logo and images editing functional by end of week 11
    • Title and slogan editing functional by end of week 12
    • Fully functional by due date
